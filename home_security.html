Smart Home Security System

Project Overview

As part of our coursework in Electrical Engineering Design 1 (ENGI 2203) , under the guidance of Dr. Sarah Stout-Grandy, P.Eng., our team tackled the challenge of developing a home security system for a model house. Leading a group of four students, I took charge of ensuring our project's completion within the specified deadlines.

The project requirements were multifaceted. We aimed to create a home security system that would demonstrate its status through LEDs, deliver intrusion alerts via a speaker, and grant users the capability to arm/disarm alarms, modify passwords, and access system statistics. Additionally, the system had to be designed to set up within a concise timeframe of five minutes, detect intrusions from windows or doors within one minute, and dismantle in just a minute. It's worth noting that our project was constrained to utilizing only the basic materials provided in our team's four design kits. The completion deadline for this endeavor was set for May 29th, 2023.

Assumptions and Constraints

For testing and design purposes, it was assumed that intruders would enter through only one entry point in the house at a time. Furthermore, due to the limited resources available, this project was constrained to the materials available in the 4 design kits provided. These materials include basic sensors and peripherals to allow for the basic functionality of the home security system.

Finalized Circuit Design

Sensor Combination Selection

Throughout the first week of the system testing and integration stages, multiple sensor combinations were tested in order to come up with the most effective method for quick and accurate intrusion detection. The results from the tests conducted are summarized in the table below:

Sensor Combination Pros Cons
2x PIR Motion Sensors on opposing sides of the house The combination was easy to setup and covered the entire house. Falsely triggered when there is motion outside the house or slight pushes on the locked door. Long calibration time
1x PIR Motion Sensor on the door and 1x Ultrasonic Sensor pointed towards the window. Accurately detected window opening through Ultrasonic Sensor. Falsely triggered from slight door pushes, even with PIR sensor sensitivity adjustments.
1x Ultrasonic Sensor on the door and 1x Hall Effect Sensor on window. Accurately detected window and door opening. Lengthy setup time.
1x Hall Effect Sensor on door and 1x Hall Effect Sensor on window. Quickly and accurately detected door and window opening. Falsely triggers on slight door pushes if placed incorrectly.

After careful consideration of the sensor test results, it was decided that the most optimal method for detecting intrusions would be achieved using two hall effect sensors placed on the door and window. This was the case, due to the quick setup time of the sensors and accurate results.

Integration and Testing in Model House

With a finalized circuit design synthesized and a user interface developed, the home security system was built then integrated into the model houses for a series of rigorous testing trials. The test conditions and outcomes of the different trial runs are summarized in the table below:

Trial Conditions Outcomes
1 Using model house with no roof, sensors placed at random locations on entry points, and two simple intrusion attempts through door and window respectively. System accurately triggered on intrusion. However, sometimes falsely triggered on door pushes.
2 Using model house with no roof and sensor placed near door hinge. No false triggers were observed. System triggered quickly and accurately on intrusion.
3 Using model house with roof added after system arming. False trigger occurred due to house morphing during roof placement.
4 Using model house with roof added after system arming, and sensor placed closer to door hinge. No false triggers were observed. System triggered quickly and accurately on intrusion.
